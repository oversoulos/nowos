# Pick a card.

**Nowos** — An experimental, decentralized card-based sandbox for creators.

</div>

## What is this?

An infinite grid of customizable cards. Each card is yours to build on. No middlemen. No algorithms. No subscriptions.

## Current Status

**Foundation phase** — Building the core interface.

- ✓ Swipeable card navigation
- ✓ Infinite grid system
- ⏳ Everything else (TBA)

## The Idea

You own your coordinates. You build what you want. It's a sandbox.

Future features include community tokens, decentralized hosting, and tools for creators. But we're starting simple.
